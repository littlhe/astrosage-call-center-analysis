# astrosage-call-center-analysis
Data analysis project analyzing AstroSage’s call center performance using Excel dashboards and insights on efficiency, customer satisfaction, and revenue.

This project analyzes AstroSage’s call center performance using historical data to evaluate operational efficiency, customer satisfaction, and revenue impact.

## Objective
AstroSage received a ₹1 crore investment and aimed to optimize its call center operations. The goal of this analysis was to identify key operational bottlenecks and provide data-driven recommendations to improve performance and profitability.

## Files in this Repository
- **AstroSage Analysis Data and Dashboard.xlsx**  
  Contains cleaned data, pivot tables, KPIs, and an interactive dashboard.

- **AstroSage Analysis PPT.pdf**  
  Presentation summarizing insights, trends, and business recommendations.

- **AstroSage Analysis Task Questions.docx**  
  Detailed analysis answers, methodologies, and explanations.

## Key Insights
- Only ~40% of total calls were successfully completed, indicating efficiency gaps during peak hours.
- Call consultations contribute the majority of revenue, making call success critical.
- Workload distribution across Astro Gurus is uneven, leading to burnout risk.
- Chat consultations show better customer satisfaction compared to calls.

## Tools Used
- Microsoft Excel (Pivot Tables, KPIs, Dashboard)
- Data Analysis & Visualization
- Business Insight & Recommendation Framework

## Outcome
The analysis provides actionable recommendations focused on technology upgrades, workforce optimization, and increased adoption of chat-based consultations to improve operational efficiency and customer satisfaction.
